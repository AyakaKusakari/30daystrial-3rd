# WordPressの自作テーマ（ブログテーマ）

「東京フリーランス」 のプログラミング学習のオンラインコミュニティに所属しており、静的HTMLサイトをWordPress化する課題に挑戦しました。

## ポイント

- PHP、SQLの基礎を一通り学習
- ローカル開発環境構築〜本番環境への移行
- トップページに新着記事の一覧表示
- トップページにピックアップ記事の表示
- アーカイブページ作成
- 投稿ページ・固定ページ作成
- ページネーションなどの共通部分は個別テンプレートに分けて管理
- 毎日の学習記録をTwitterで発信
- コミュニティ内で質問したり、他の方の質問に答えたりして助け合った

## 使用言語
HTML5 / CSS3 / jQuery / PHP
